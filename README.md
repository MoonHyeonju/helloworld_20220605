2)vim 에디터에서 매크로 사용방법

1- 명령어 모드에서 'q'와 a ~ z 사이 문자를 입력해서 매크로 recoding 시작

2- 기록이 끝나면 다시 'q'를 입력

3- '@' + a ~ z를 입력해서 매크로 재생

4- 매크로 저장 방법:  ~/.vimrc파일을 열어서 let @문자 = '매크로 문자열' 입력
